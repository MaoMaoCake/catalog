## About SapMachine

SapMachine is SAP's downstream distribution of OpenJDK, providing production-ready JDK and JRE builds. The DHI
sapmachine image packages the SapMachine JDK in a minimal container with JAVA_HOME set to /opt/java/openjdk and common
Java CLI tools available on PATH, making it suitable for running Java applications and as a base image for multi-stage
builds.

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted

### Why use Docker Hardened Images?

These images are published with near-zero known CVEs, include signed provenance, and come with a complete Software Bill
